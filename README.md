## Getting Started

### copy

```
git@github.com:zainsadaqat/hello-rails-front-end.git
```

####

```
git clone git@github.com:zainsadaqat/hello-rails-front-end.git
```

```
cd hello-rails-front-end
```

```
bundle install
```

```
rails s
```
